# Banking System With Spring Boot And Angular with Spring Security

## Introduction

Following Functionalities Implemented
- Account creation
- Transactions between two accounts
- Fetching of the transaction history
- Register Customers

## Prerequisites
- java 1.8.x
- maven 3.x
- npm

### Steps To Setup Backend

**1. go to root directory of backend**

**3. Build project**
```bash
    mvn clean install
``` 

**4. Run project** 
```bash
    mvn spring-boot:run
```

  #### Explore apis in backend

The app defines following APIs. 
 
```   
    POST /api/signup   
    POST /api/user
    GET /api/user
    PUT /api/user
    GET /api/account
    POST /api/transfer
    GET /api/transaction      
```

### Steps To Setup Frontend

**1. Move To Frontend Derectory**

**2. Install Package**
```bash 
    npm install
```

**3. Run Project**
```bash
    npm start
```

**4. Open url**
```bash
    http://localhost:4200/
```
